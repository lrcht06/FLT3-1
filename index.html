Finals Lab Task 3-1

MySQL SELECT Clauses Portfolio

About

This portfolio demonstrates the application of MySQL SELECT Clauses to query and retrieve data from a relational database. The task builds upon a predefined database structure and focuses on extracting meaningful insights using various SELECT statement features.

Step-by-Step Process

  1. Launch MySQL Workbench or phpMyAdmin and start XAMPP to activate MySQL.
  2. Import the onlineCourse.sql file to initialize the tables and data.
  3. Create a database named online_courseDB and use it.
  4. Run the provided SQL structure to load the tables and sample data.
  5. Execute the following SELECT queries:
  • Task 1: Retrieve courses where students enrolled < enrollment limit.
  • Task 2: Group by category, calculate total students enrolled.
  • Task 3: Retrieve fully enrolled courses.
  • Task 4: Calculate total students across all courses.
  • Task 5: Sort courses by students enrolled (ascending).
